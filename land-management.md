Update the **Land Management Page** and **Add Land Record Modal** by adding these important fields:

### Land Management Table Add Columns

* Land ID / Land Code
* Record Type (CS/SA/RS/BRS)
* District
* JL No.
* Land Type
* Share (%)
* Acquisition Type
* Acquisition Date
* Mutation Status
* Development Agreement Status
* Total Owners
* Available Area

### Add Land Record Modal Add Fields

Create new sections:

**Land Identification**

* Land ID / Land Code
* Record Type (CS/SA/RS/BRS)
* District
* JL No.

**Land Details**

* Land Type
* Share (%)
* Available Area

**Acquisition Information**

* Acquisition Type
* Acquisition Date

**Legal & Ownership Information**

* Mutation Status
* Development Agreement Status
* Total Owners

Make sure these fields are available in both the table view and add/edit land record form.
